# User List

## implementation of user cards with user authorization

- SPA;
- Redux + Redux Toolkit;
- Webpack for frontend;
- MUI;
- SCSS module (bootstrap, CSS animations);
- Pagination + page loading implementation;
- Custom form validation;
- Firebase;
